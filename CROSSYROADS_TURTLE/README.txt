# 🐢 Turtle Crossy Road - Basic Version (OOP)

A simple Crossy Road–style game built using Python's Turtle graphics and Object-Oriented Programming (OOP).

---

## 🎮 How to Play

- Press the **Spacebar** to move the turtle (player) forward.
- Avoid the **multicolored cars** moving horizontally across the screen.
- Reach the **top of the window** to progress to the next level.
- With each new level, the **speed of cars increases**.
- **Getting hit by a car = Game Over!**

---

## 🚀 Features

- Built using clean Object-Oriented Programming (OOP) design
- Multiple levels with increasing difficulty
- Colorful visuals and simple controls

---

## 🛠 Requirements

- Python 3.6+
- Turtle module (comes pre-installed with Python)

---
